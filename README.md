# 🚀 Data-Home Light - Landing Page

Ceci est une landing page moderne et ultra-rapide conçue spécifiquement pour les **agences immobilières**. 

L'objectif de ce projet est de maximiser la conversion des visiteurs en mandats grâce à une performance technique irréprochable (chargement en moins d'une seconde).

## 🌍 Aperçu en direct
[https://datahome.vercel.app/en/schmidt-privilege]

## ✨ Points forts
- **Performance :** Score Google PageSpeed > 95.
- **Design Premium :** Interface moderne, épurée et responsive (mobile-first).
- **Conversion :** Structure optimisée pour la prise de contact rapide.
- **Mise en service :** Déploiement en 48 heures.

## 🛠️ Technologies utilisées
- **HTML5** : Structure sémantique.
- **Tailwind CSS** : Framework CSS utilitaire pour un design rapide et flexible.
- **Google Fonts** : Utilisation de la police 'Inter' pour une lisibilité maximale.
